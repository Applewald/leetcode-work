# JavaLeetcodeWork
### 介绍
Java语言刷Leetcode

### 基本思路
1. 刷前准备
    - 经典排序算法（快排、归并、堆排序）
    - 经典遍历算法（层次、前序、后序，遍历算法可以不必刷前准备，遇到再写）
2. 开始刷题
    - 先刷简单、中等难度的题，简单优先自行实现，中等稍微尝试一下，不要耽误太长时间，想不出来就抄答案
    - 直接按顺序刷，在前20题里找，以上手习惯
3. 进阶
    - 上手之后，寻找不错的题目单开始刷题，优先排序、查找等基本算法
    - 对自己水平有一定了解之后，自己用标签筛选功能找题目刷
4. 可持续
    - 每周至少刷5天，至少刷7题
    - 每天最少一道题，最多3题，题目过多会导致抗拒
